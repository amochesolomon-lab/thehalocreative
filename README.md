# Sol'o Mon

React + Vite portfolio site for Sol'o Mon with Framer Motion animations and an editorial luxury layout.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Structure

- `src/` contains the React app, pages, reusable UI, and site data.
- `assets/` keeps the original portfolio imagery.
- Legacy static files remain in the repo for reference, but the React app is the active site.
